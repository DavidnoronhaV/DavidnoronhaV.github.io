Cookie Forms , Data Saved In The Browser



How It Works

When A User Inputs Data Into A Cookie Form The Data Is Saved As A Javascript Cookie Along With The User's IP Address. So In Other Words, The User's Input Is Only As Safe As His Computer It-Self. When A User Wants To Use That Infomation On The Same Website Where He Saved It, The Website Can Easily Use It And As A Cookie Is Reserved Only For A Certain Website.
